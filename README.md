Hospital Stay Prediction

A machine learning project that predicts the estimated hospital stay duration of patients based on medical and demographic information. This project helps hospitals improve resource planning and patient management.

Project Overview

The goal of this project is to build a predictive model using machine learning algorithms to estimate how long a patient may stay in the hospital.
